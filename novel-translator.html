Client ID ဘယ်လိုရမလဲ (Setup Guide)
  1. console.cloud.google.com ကို ဖွင့်ပါ၊ Project အသစ်တစ်ခု ဆောက်ပါ
  2. "APIs & Services" → "Library" → "Google Drive API" ကို ရှာပြီး Enable လုပ်ပါ
  3. "APIs & Services" → "Credentials" → "Create Credentials" → "OAuth client ID"
  4. Application type: Web application ရွေးပါ
  5. "Authorized JavaScript origins" မှာ ဒီစာမျက်နှာကို ဖွင့်နေတဲ့ URL ကို ထည့်ပါ (ဥပမာ https://claude.site သို့ ဖိုင်ကို ဖွင့်ထားတဲ့ origin)
  6. Client ID ကို ကူးပြီး အပေါ်က field ထဲ ထည့်ပါ
  7. "APIs & Services" → "OAuth consent screen" မှာ test user အနေနဲ့ မိမိ Google account ကို ထည့်ထားရပါမယ် (App က verify မလုပ်ထားသေးရင်)
